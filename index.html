Community guidelines

  • Spam, Selling & Advertising
    Kylee's World does not allow Spam, Selling nor Advertising Especially in People's DMS. Those who broke them will receive consequences 
  • User Rules
    -- No inappropriate nicknames
    -- No sexually explicit nicknames
    -- No offensive nicknames No nicknames with unusual or unreadable Unicode
    -- No sexually explicit profile pictures
    -- KEEP IT SFW
  • Text Chat Rules
    -- No mentioning anyone ( without their permissions )
    -- No sexually explicit content
    -- Do not talk about an activity which is disturbing for some
    -- Keep it SFW
    -- Do not Mini Mod ( Do >report if a suspicious activity is suspected )
  • Voice chat rules
    --     No voice chat channel hopping
    -- No annoying, loud or high pitch noises
    -- Reduce the amount of background noise ( if possible )
    -- Moderators reserve the right to disconnect, mute, deafen, or move members to and from voice channels.
  • Bot specific rules
    -- No Command Spam
    -- No Abusing Commands
  • Keep it SFW
    Kylee's Server is a SFW server so for that purpose please keep NSFW out of Kylee's Server
